إرشادات إنشاء صورة المشاركة:

  1. اضغط F12 لفتح Developer Tools
  2. اضغط بالزر الأيمن على الصورة أعلاه
  3. اختر "Capture node screenshot" أو "Screenshot node"
  4. احفظ الصورة باسم "og-image.png" في مجلد img
  5. غير مسار الصورة في meta tags ليشير للصورة الجديدة

أو يمكنك استخدام أدوات مثل Canva أو Figma لإنشاء صورة 1200×630 بكسل
